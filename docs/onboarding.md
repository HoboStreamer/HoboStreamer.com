# Onboarding Guide

Welcome to HoboStreamer! This guide walks you through getting set up.

## 1. Create an Account

Sign up at [hobo.tools](https://hobo.tools) to create your Hobo Network account. This single sign-on (SSO) account works across all Hobo Network services including HoboStreamer.

## 2. Choose Your Streaming Method

HoboStreamer supports three protocols:

| Method | Best For | Latency | Difficulty |
|--------|----------|---------|------------|
| **WebRTC (Browser)** | Quick streams, mobile | Sub-second | Easy |
| **RTMP (OBS)** | Professional setups | 2-5 seconds | Medium |
| **JSMPEG (FFmpeg)** | Headless/IoT devices | ~1 second | Advanced |

See the [Broadcasting Guide](broadcasting.md) for detailed setup instructions.

## 3. Go Live

1. Navigate to the **Broadcast** page
2. Fill in stream title, description, and category
3. Select your streaming method
4. Click **Create Stream**
5. Follow the protocol-specific instructions

## 4. Explore Your Dashboard

Your dashboard (accessible when logged in) includes:

- **Stream Key** — For RTMP/OBS setup
- **API Tokens** — Create tokens for bots and integrations
- **Chat Logs** — View and manage your stream's chat history
- **Emotes** — Upload custom channel emotes
- **VODs & Clips** — Manage recordings and clips
- **Hobo Bucks / Nickels** — Currency and rewards system
- **Coin Rewards** — Create viewer rewards

## 5. Chat Features

- **Popout Chat** — Open chat in a separate window for multi-monitor setups
- **TTS (Text-to-Speech)** — Configurable voice reading of chat messages
- **Emote Picker** — Custom and third-party emotes (FFZ, BTTV, 7TV)
- **GIF Search** — Inline GIF support via Tenor
- **Chat Replay** — Messages are stored and replay alongside VODs

## 6. Broadcast Controls

While streaming via WebRTC, you have access to:
- Camera switching and screen share
- Live clipping
- Viewer-style preview popup
- Disconnect sound alerts (opt-in)
- Real-time stats overlay
- TTS controls

## 7. VOD System

Streams are automatically recorded. Features include:
- **DVR Seeking** — Viewers can rewind live streams
- **Automatic Thumbnails** — Generated from recording content
- **Privacy Controls** — Set VODs as public or private
- **Bulk Management** — Delete old media by age
